#  M31 -  TP Outils informatiques

## Livrable

Le compte rendu de ces TPs, rédigé à l'aide de la suite OpenOffice ou de LibreOffice, sera obligatoirement déposé en fin de séance sur la plate-forme moodle du module M23.
